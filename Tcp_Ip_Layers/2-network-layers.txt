TCP/IP NETWORK LAYERS
---------------------
1. Application Layer (DATA):
   - User-facing apps (Browser, Outlook).
   - Protocols: HTTP, FTP, SMTP.
2. Transport Layer (SEGMENT):
   - TCP adds ports/sequence numbers.
   - Ensures error-free delivery.
3. Network Layer (PACKET):
   - Adds source/destination IPs.
4. DataLink Layer (FRAME):
   - Adds MAC addresses + error checks.
   - MAC = Device's unique hardware ID.
5. Physical Layer:
   - Hardware (cables, signals).
