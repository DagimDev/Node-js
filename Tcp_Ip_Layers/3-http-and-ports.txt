HTTP & WEB SERVERS
------------------
1. HTTP Protocol:
   - Transfers text/files over TCP/IP.
   - Uses port 80 by default.
2. Web Servers:
   - Software (e.g., Apache, Nginx).
   - Listens on port 80 for HTTP requests.
   - Sends static files directly; processes dynamic content.
